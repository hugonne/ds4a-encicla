Encicla Insights

Encicla, the public bike-sharing system of the Aburra Valley, inaugurated in 2011, is a network of shared bicycle stations for the mobilization of people in the Metropolitan Area.

Learn more »

Business Problem

Giving insights about a dataset is the aim of each data science project, in this specific project route optimization and demand predicting could help Encicla in:

  • Proper Planning and cost optimization routes for bicycles supply.
  • Customer/users Optimal offers according with available options.
  • Increasing the accuracy of pre-decided arrival time of bicycles to Encicla Stations.
  • Delivering more in less time and minor distance.
  • Improved customer assistance experience through visualization availability.
  • Accurate ETA showing planned routes and possible stoppages for bicycles supply.

Data

The dataset consists of public information about mobility of Medellin:

  • Location of the Encicla public bike-sharing system stations
  • Rest API Service with real time data about inventory of bike station

Methods

One key component of understanding the patterns in Encicla's bicycle usage is providing high-quality visualizations of his historical user's interactions with Encicla's system. Here are some of the static and interactive visualizations we will provide:

  • Encycla's bicycles usage over time (historical usage by weekdays-hours and week-days)
  • Inventory nominal value against real value (+ 1 static plot with interactions over stations)
  • Optimal Route supply visualization (+ 1 static plot with interactions over stations)
  • Performance metrics of each model vs reality (Back testing metrics)
